# База данных репозитория

Консольная программа для работы с файловой базой данных репозиториев.

## Автор

Костопраев Дмитрий Евгеньевич

## Структура проекта

```
repository.h      - заголовочный файл (структуры, прототипы)
repository_db.c   - функции работы с БД
io.c              - функции ввода-вывода
main.c            - главный модуль
data.txt          - пример файла данных
```

## Сборка

```bash
gcc -std=c99 -Wall -o repository.exe main.c repository_db.c io.c
```

## Запуск

```bash
repository.exe
```

## Формат файла данных

Каждая запись - 7 строк:

```
<Направление>
<Сайт>
<Название>
<Размер>
<День> <Месяц> <Год>
<Зависимости>
<Совместимость>
```

Направление: Backend, Frontend, Mobile, DevOps, DataScience

Совместимость: Windows, Linux, macOS, CrossPlatform

## Пример записи

```
Backend
https://github.com/example/plugin
AuthPlugin
256
15 3 2023
5
CrossPlatform
```

## Функции программы

1. Загрузить из файла
2. Просмотреть записи
3. Поиск по направлению
4. Комбинированный поиск (дата + размер)
5. Сортировка (пузырьковая)
6. Добавить запись
7. Сохранить в файл
8. Выход

## Сортировка

Алгоритм: пузырьковая сортировка (Bubble Sort)

Критерий: Название (возр.) -> Направление (возр.) -> Дата релиза (убыв.)
